## Запуск приложения

### Установка и запуск
```shell
git clone git@github.com:yurylunev/middle.messenger.praktikum.yandex.git
npm install
npm start
```

http://localhost:3000

### Запуск версии для разработчиков
```shell
npm run dev
```
http://localhost:1234

### Сборка рабочей версии приложения
```shell
npm run build
```
